# Agentic CMS for JVM

The CMS your AI runs.

## What is Agentic CMS

Agentic CMS is an open-source, AI-first CMS you run from your AI assistant. It ships a built-in MCP server, so Claude Desktop, ChatGPT, or any MCP client can manage the whole site through a typed, permissioned interface, by chat instead of clicking through an admin panel: content, media, roles, moderation, SEO.

## AI capabilities

- Write and edit content
- Moderate comments
- Semantic search
- RAG-ready answers

## This edition

Agentic CMS for JVM targets the Java / JVM stack with PostgreSQL. It's part of a family of editions, each fast and native to its own stack.

**Status: Planned / early access.** This edition is not shipped yet. Visit the [landing page](https://elman.group/agentic-cms/) to get notified when it's available.
